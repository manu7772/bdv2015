load

Parameters
{String}url Absolute URL to script to add.
{function}callback Optional callback function to execute ones this script gets loaded.
{Object}scope Optional scope to execute callback in.

Returns true/false if a script has been loaded or not.

isDone

Parameters
{String}url URL to check for.
Returns
{Boolean} true/false if the URL is loaded.

Marks a specific script to be loaded. This can be useful if a script got loaded outside the script loader or to skip it from loading some script.

markDone

Parameters
{string}u Absolute URL to the script to mark as loaded.

Adds a specific script to the load queue of the script loader.

add

Parameters
{String}url Absolute URL to script to add.
{function}callback Optional callback function to execute ones this script gets loaded.
{Object}scope Optional scope to execute callback in.

Starts the loading of the queue.

loadQueue

Parameters
{function}callback Optional callback to execute when all queued items are loaded.
{Object}scope Optional scope to execute the callback in.

Loads the specified queue of files and executes the callback ones they are loaded. This method is generally not used outside this class but it might be useful in some scenarios.

loadScripts

Parameters
{Array}scripts Array of queue items to load.
{function}callback Optional callback to execute ones all items are loaded.
{Object}scope Optional scope to execute callback in.
